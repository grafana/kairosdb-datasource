Auto Value Intervals

A comma delimited list of intervals used for auto value sampling in grafana time notation (e.g. 1m,1h)

This will cause 'Auto' sampled aggregators to select the closest interval bigger than grafana's provided interval for the time range selected
Require Scalar Aggregators

This setting allows for enforcement that queries include at least one scalar aggregator per query.

This option should be enabled when most data is histograms that need to be converted into scalars in order to be graphed in common visualizers, e.g. timeseries line graphs
